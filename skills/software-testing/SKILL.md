## Best Practices I Follow

1. **Test Independence**: Each test should be independent
2. **Clear Naming**: Test names should describe what they test
3. **AAA Pattern**: Arrange, Act, Assert
4. **Fast Feedback**: Tests should run quickly
5. **Comprehensive Coverage**: Test happy paths and edge cases
6. **Mocking**: Mock external dependencies
7. **Flaky Tests**: Identify and fix flaky tests
8. **Test Data**: Use realistic test data

## Commands I Use

```bash
# Run tests
pytest                      # Python
npm test                    # JavaScript
mvn test                    # Maven
go test ./...               # Go

# Coverage
pytest --cov=src            # Python
npm run coverage            # JavaScript
mvn jacoco:report           # Maven

# E2E tests
cypress run                 # Cypress
selenium-side-runner        # Selenium
```
